# App Icon Redesign Implementation Plan

> **Retrospective backfill.** This plan documents work that has already shipped in commit `79cd8d3` (2026-05-13). It is recorded here so the specs/plans index stays consistent. Step checkboxes are marked done where the corresponding work landed; the one outstanding item (full visual verification across every surface) is left unchecked.

**Spec:** [2026-05-13-app-icon-design.md](../specs/2026-05-13-app-icon-design.md)

**Goal:** Replace the generic two-circle logo with a distinctive flat clipboard-slip + dog-eared corner logomark. One hand-authored SVG master drives every derived asset (tray icon and its paused variant, the bundled `.ico`/`.icns` app icon, the favicon, and every Microsoft Store `Square*Logo` tile). The earlier "single bold letter Y" concept (spec [2026-05-07-statusbar-icon-design.md](../specs/2026-05-07-statusbar-icon-design.md)) was abandoned after evaluating it against Microsoft's Windows app-icon design guidance, which explicitly recommends against typography in app icons; that spec is marked superseded.

**Tech stack:** Tauri 2 icon CLI (`pnpm tauri icon`), `sharp` (devDependency) for SVG rasterization and the paused-variant pixel walk, hand-authored SVG.

## Task 1: Ship the new icon pipeline end-to-end

**Files:** all of the above. Single squashed commit `79cd8d3 feat(icon): replace app icon with flat clipboard-slip dog-ear design`.

- [x] **Step 1: Author the SVG master**
  Commit `src-tauri/icons/icon-source.svg` with the geometry described in the spec: rounded slip (~70% × ~78% of canvas, ~17% corner radius), diagonal cut from the top edge (65% across) to the right edge (30% down), reflected flap exposing the paper underside, darker-teal crease along the fold, 3° counter-clockwise rotation. Palette: `#0F8FA0` slip / `#FFC72C` flap / `#0A6E7C` crease / transparent background.

- [x] **Step 2: Rasterize the SVG to a 1024×1024 PNG master**
  ```bash
  node -e "require('sharp')('src-tauri/icons/icon-source.svg',{density:288}).resize(1024,1024).png().toFile('src-tauri/icons/icon-source.png')"
  ```
  Output: `src-tauri/icons/icon-source.png` (32 KB, RGBA, transparent background).

- [x] **Step 3: Regenerate every Tauri-managed variant**
  ```bash
  pnpm tauri icon src-tauri/icons/icon-source.png
  ```
  Produces `icon.png`, `icon.ico`, `icon.icns`, the 32/64/128 PNG ladder, and every `Square*Logo` tile.

- [x] **Step 4: Derive the paused tray variant**
  Add `scripts/generate-paused-icon.mjs` that loads `src-tauri/icons/32x32.png`, replaces every non-transparent pixel with `#9CA3AF` while preserving the alpha channel, and writes `src-tauri/icons/32x32-paused.png`. Run once:
  ```bash
  node scripts/generate-paused-icon.mjs
  ```

- [x] **Step 5: Sync the favicon**
  ```bash
  cp src-tauri/icons/icon.ico public/favicon.ico
  ```

- [x] **Step 6: Update the tray tooltip**
  In `src-tauri/src/lib.rs`, change the `TrayIconBuilder` `.tooltip(...)` call from `"Clipboard Manager"` to `"YANK"` (matches `productName` in `tauri.conf.json`).

- [x] **Step 7: Verify the brand colours rendered correctly**
  Center-pixel sampling on the generated assets confirmed:
  - `32x32.png` and `icon.png` → `#0F8FA0` (slip teal)
  - `32x32-paused.png` → `#9CA3AF` (neutral grey)

- [x] **Step 8: Commit**
  ```bash
  git add src-tauri/icons/ scripts/generate-paused-icon.mjs public/favicon.ico src-tauri/src/lib.rs package.json pnpm-lock.yaml docs/superpowers/specs/
  git commit -m "feat(icon): replace app icon with flat clipboard-slip dog-ear design"
  ```

- [ ] **Step 9: Visual verification across every surface (pending)**
  Run `pnpm tauri build` and confirm by eye: tray icon active + paused, Start menu tile, taskbar entry, installer header, favicon in the browser when running `pnpm start`. Flag any surface where the icon reads muddy at small sizes (especially the auto-downsampled 16×16 ICO frame) for a follow-up.

## Out of Scope (per spec)

- Animated or motion variants on the tray icon.
- Per-locale or per-theme alternate marks (deferred until observed contrast issues against light Windows accent colours).
- A hand-tuned bespoke 16×16 PNG frame for the tray ICO — flag separately if the auto-downsampled frame looks muddy in the notification area.
- Updating in-app illustrations or empty-state graphics.
